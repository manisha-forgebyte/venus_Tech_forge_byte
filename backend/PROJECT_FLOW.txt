==================================================
5. COMMON STRUCTURE
==================================================

src/common/

Reusable global backend components.


--------------------------------------------------
decorators/
--------------------------------------------------

Special reusable NestJS behaviors.

Files:

current-user.decorator.ts
Gets logged-in user data.

public.decorator.ts
Marks API as public.

roles.decorator.ts
Defines allowed roles.


Concept:
Declarative Programming


--------------------------------------------------
filters/
--------------------------------------------------

Global error handling.

Files:

all-exceptions.filter.ts
Handles unexpected errors.

http-exception.filter.ts
Handles HTTP errors.


Concept:
Centralized Exception Handling


--------------------------------------------------
guards/
--------------------------------------------------

Protect APIs.

Files:

jwt-auth.guard.ts
Checks JWT token.

roles.guard.ts
Checks user roles.


Concepts:
Authentication
Authorization


--------------------------------------------------
interceptors/
--------------------------------------------------

Runs before/after requests.

Files:

logging.interceptor.ts
Logs API activity.

response.interceptor.ts
Formats responses.

timeout.interceptor.ts
Stops long-running requests.


Concept:
Cross-Cutting Concerns


--------------------------------------------------
middlewares/
--------------------------------------------------

Runs before controller.

Files:

logger.middleware.ts
Request logging.

request.middleware.ts
Preprocess requests.


Concept:
Request Lifecycle


--------------------------------------------------
pipes/
--------------------------------------------------

Validates & transforms data.

Files:

parse-int.pipe.ts
Converts string to number.

trim.pipe.ts
Removes extra spaces.

validation.pipe.ts
Validates request data.


Concept:
Input Validation & Sanitization


--------------------------------------------------
utils/
--------------------------------------------------

Reusable helper functions.

Files:

date.util.ts
Date helpers.

response.util.ts
Standard API responses.

token.util.ts
JWT token helpers.


Concept:
DRY Principle
(Don’t Repeat Yourself)
